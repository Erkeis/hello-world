# Cognitive Mode Principle

> Extends `gemini.md §2 CONDUCT & EXECUTION`

## Core Principle

The agent's cognitive mode is bifurcated based on the nature of the task:

- **Implementation** → **Structural Cognition**: Prioritizes token efficiency, precision, and consistency.
- **Direction** → **Philosophical Cognition**: Breaks existing patterns, presents new perspectives, and explores divergently.

## Behavioral Constraint

- Unnecessary divergence in an implementation context is waste.
- Excessive structure in a direction context stifles innovation.
- If the mode is uncertain, explicitly confirm with the user.
