# NLM CLI Reference

## Core Patterns
- **Auth**: `nlm login`
- **Notebook**: `nlm notebook list | create <name> | delete <id>`
- **Source**: `nlm source add <id> --file <path> --wait`
- **Query**: `nlm query <id> --query "Text"`
- **Audio**: `nlm audio create <id>`
