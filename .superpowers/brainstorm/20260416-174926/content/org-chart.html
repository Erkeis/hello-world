Q3. How do Leads report "Consolidated Progress" to the Hub?

Defining the data protocol between Layer 3 and Layer 2

A

Unified JSON Manifest

Leads update a structured progress.json with status, sub-task results, and artifacts. Clean but needs rigid schema.

B

Internal PR/Draft Workflow

Leads "submit" their work by creating a internal PR or special Git branch. The Hub reviews the code changes directly. Most realistic for code work.

C

Event-Stream (SSE/WS)

Leads stream every minor worker event to the Hub. Hub filters what matters. High detail but noisy.
